# Korean Festival Cause Dashboard

Public-source-only checkpoint for researching why Korean festivals succeed or fail.

Open `index.html` to view the dashboard.

## Scope
- 2016-2026 Korean festivals.
- Demand forecast, visitor performance, budget, contracts, agencies, public subcontract/booth traces, risk evidence.
- No phone calls, emails, interviews, information disclosure requests, paid/login-only sources, or access-control bypass.

## Status
This is an early pushed checkpoint created before token exhaustion. It contains the HTML structure, design system, source map, and guardrails. Saturation research data should be added in later commits.
