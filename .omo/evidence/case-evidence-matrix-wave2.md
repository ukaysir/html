# Case evidence matrix wave 2

Generated: 2026-07-08.

Purpose: make the growing dossier set readable as a cause triage table. The user question is not "which festival page exists?" but "why did this fail or succeed, and is it contractor, booth, demand, budget or another cause?"

Interpretation rules:
- Contractor/subcontractor causality requires outcome evidence, public procurement/contract scope, timing match, and a plausible mechanism. Procurement evidence alone is not fault evidence.
- Price/booth cases are split into official vendor, nearby/night-market seller, broker/seat-fee lead, and unverified subcontractor path.
- Demand/crowd cases are split into expected attendance, legal/safety planning threshold, bottleneck, deployment timing, traffic impact and satisfaction signal.
- Budget waste is never inferred from high budget alone. It needs public cost, public value, local capture, external cost and accountability mismatch.
- Success cases remain useful because they show which controls can prevent failure modes.
