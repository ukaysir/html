# korea-festival-agent-only-research - Work Plan Checkpoint

## Objective
Execute public-source-only research for 2016-2026 Korean festivals and produce a human-readable HTML dashboard focused on why festivals succeeded or failed.

## Execution guardrails
No phone calls, emails, interviews, information disclosure requests, paid/login-only sources, or access-control bypass. Hidden subcontract/vendor claims must not be asserted without public evidence.

## Research priority
1. Official festival master sources.
2. Tourism visitor/spending/SNS sources.
3. Local finance cost/accounting/budget sources.
4. Procurement and contract sources.
5. Public vendor, subcontract, booth, merchant traces.
6. Official/council/audit/news issue evidence.
7. Public reviews/SNS/blogs only as lower-confidence sentiment signals.

## HTML requirement
The HTML must show diagnosis first and raw data last. Required user path: symptom -> likely cause -> stakeholder/contract relation -> evidence -> source.
